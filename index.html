Praticando com o Git e Github

O que é o GIT?

O GIT é um sistema de versionamento de códigos que possui 3 estados, sendo eles: modificado, preparado e consolidado. No caso, ele cria cópias de alterações e branches (caminhos) para tornar mais simples o desenvolvimento de códigos.

O que é o Github

O GitHub é uma espécie de "rede social" de desenvolvedores, onde é possível publicar ou encontrar códigos de diversas linguagens, possibilitando o compartilhamento de informações. É um tipo de repositório online.
